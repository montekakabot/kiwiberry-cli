We want to build a CLI tool to help business to keep track of their Yelp reviews. If there is new review, we will help them to draft responses. This CLI tool is going to use with OpenClaw Skills. We will need to use [Openclaw brwoswer CLI](https://docs.openclaw.ai/cli/browser) to go to business's yelp page to scrape reviews. Our CLI tool would use SQLite3 to store data. We will use Typescript, and Bun.js. Our CLI tool should be distribute with a built image.
